Doxr Toolkit Documentation

This is the doxr toolkit.

You can use it like this:

<script src="https://toolkit.doxr.repl.co/kit.js"></script>

This applies to mainly vanilla JS and it's to make Javascript easier. You will find types; SS stands for "Send Stuff;" it returns things but doesn't do anything by itself. Type DS stands for "Do Stuff;" it does stuff, and it's like a normal function that makes JS simpler.

element(id)

The equivalent to document.getElementById(id); this uses an element's ID to get an element. Regard the next function for classes.

elementC(class)

The equivalent to document.getElementByClassName(class); this uses an element's class to get an element.

writeTo(id, content)

The equivalent to document.getElementById(id).innerHTML = content; this uses an element's ID to change it's content. The next function uses it's class instead.

writeToC(class, content)

The equivalent to document.getElementByClassName(class).innerHTML = content; this uses an element's class to change it's content.

style(id, styletype, stylevalue)

The equivalent to document.getElementById(id).style.styletype = stylevalue; this uses an element's ID to change it's style. styletype is what style is being changed and stylevalue is styletype's value.
